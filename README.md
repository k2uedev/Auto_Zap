# Auto Zap - WhatsApp

## Descrição
Este projeto permite o envio de mensagens em massa via WhatsApp, utilizando uma interface gráfica moderna e minimalista, construída com HTML, CSS e JavaScript, que eu recomendo para empresas e marketeiros que precisam enviar mensagens em massa mas não querem perder tempo enviando de um a um ou, gastar dinheiro com algumas ferramentas na web.

## Funcionalidades
- Envio de mensagens em massa para números de WhatsApp especificados.
- Definição de número de repetições e intervalo entre envios.
- Opções adicionais:
  - Confirmar envio antes de iniciar.
  - Notificar após cada envio.
  - Salvar histórico de mensagens enviadas.

## Como Usar

1. **Abrir o Arquivo**:
    - Em breve tera o GITHUB.io para testes

2. **Preencher o Formulário**:
    - Insira o número do WhatsApp no formato internacional e completo, usando o '+' (ex: `+5511987654321`).
    - Escreva a mensagem que voce quer enviar.
    - Escolha quantas vezes a mensagem será enviada.
    - Defina o intervalo entre os envios.
    - Se precisar, selecione as opções adicionais - ainda não funcionais por ser um modelo incial.

3. **Enviar Mensagens**:
    - Clique no botão "Enviar" para começar a enviar as mensagens.

## Notas
- O WhatsApp Web precisa estar logado no navegador usado.
- Cada mensagem será aberta em uma nova aba do navegador. Dependendo das suas configurações, você pode precisar confirmar o envio manualmente.
